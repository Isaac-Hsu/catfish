Combines computer vision and automation. Detects on-screen elements (such as fish or hazards), makes decisions based on predefined logic, and performs actions accordingly. 
Focuses on:
Image recognition
Decision-making systems
UI interaction workflows
Simulated human-like input patterns

# Features
Image Detection: Identifies relevant on-screen elements (e.g., targets vs hazards).
Decision Logic: Determines whether to interact with detected objects based on defined criteria.
Automated UI Flow: Handles interface actions such as retrying or continuing after events.
Error Handling: Recovers from failed interactions or unexpected UI states.
Input Simulation: Uses probabilistic timing for more natural interaction patterns.
